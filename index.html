Projects

  • Protein-inspired 3D Diffusion
    A generative model using diffusion transformers and reinforcement learning to design 3D protein-like chain structures. Trains a transformer-based DDPM/DDIM denoiser and fine-tunes it with DDPO policy gradients to optimize structural properties such as beta-sheet alignment.

  • 2D Diffusion
    An implementation of DDPM and DDIM on 2D toy datasets (circles, two-moons, spirals) using an MLP denoiser with sinusoidal embeddings and cosine noise scheduling. Built to develop intuition for the forward and reverse diffusion process through clean visualizations.
