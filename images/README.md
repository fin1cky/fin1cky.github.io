# Images

Place your assets here:

- `profile.jpg` — your profile photo (square, ≥320×320px recommended)
- `paper1.png`, `paper2.png`, … — publication thumbnails (160×120px recommended)
- `paper1_hover.png` — optional hover image for a paper (same size as thumbnail)

The `data/` folder at the project root is for files like `cv.pdf`.
